# Быстрый старт Frontend

## Шаг 1: Установка зависимостей

```bash
cd frontend
npm install
```

## Шаг 2: Настройка окружения

Создайте файл `.env`:
```bash
cp .env.example .env
```

Убедитесь, что в `.env` указан правильный URL API:
```
VITE_API_BASE_URL=http://localhost:8000/api/v1
```

## Шаг 3: Запуск Backend

Убедитесь, что Django backend запущен:
```bash
# В корневой директории проекта
python manage.py runserver
```

Backend должен быть доступен на `http://localhost:8000`

## Шаг 4: Запуск Frontend

```bash
npm run dev
```

Frontend будет доступен на `http://localhost:3000`

## Шаг 5: Вход в систему

1. Откройте браузер: `http://localhost:3000`
2. Введите учетные данные
3. После успешного входа вы попадете на Dashboard

## Проверка работы

- ✅ Страница логина открывается
- ✅ Можно войти с существующими учетными данными
- ✅ После входа отображается Dashboard
- ✅ Статистика загружается с backend
- ✅ Навигация работает

## Возможные проблемы

### CORS ошибки

Убедитесь, что в Django settings.py добавлен `http://localhost:3000` в `CORS_ALLOWED_ORIGINS`

### Токены не сохраняются

Проверьте консоль браузера на ошибки. Убедитесь, что localStorage доступен.

### API запросы не работают

1. Проверьте, что backend запущен
2. Проверьте URL в `.env`
3. Проверьте консоль браузера на ошибки
